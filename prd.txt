PressCoach — AI Journalist Interview Simulator (PRD)

Project Name
PressCoach — AI Journalist Interview Simulator

Summary / Objective
PressCoach simulates a live journalist interview using voice input. Founders answer verbally; the system uses Gemini models to transcribe, ask follow-up questions, assess responses, and provide PR evaluation. It also generates an AI portrait of the journalist and a fictional media outlet logo.

Core User Value

Practice high-pressure interviews safely.

Improve clarity, credibility, and messaging.

Receive actionable PR feedback.

Become more media-ready.

Tech Stack

Python 3.10+

Gemini API (Gemini 2.5 Pro, Flash, Flash Audio)

Streamlit UI

Pillow for images

sounddevice/scipy for optional CLI recording

User Flows

5.1 Startup Flow

User enters startup description.

User selects persona, difficulty, and optional news context.

System generates journalist portrait + outlet logo.

Interview screen loads.

5.2 Interview Loop

Show journalist image + outlet logo.

Journalist asks Question #1.

User records voice.

STT converts audio to text.

Q&A saved.

Gemini generates next question.

8–12 questions total.

5.3 Final Evaluation

System evaluates transcript (strengths, weaknesses, scores, quotes, rewrites, headline, training plan).

UI displays evaluation.
